## 摘要
我们引入了一个受控的子空间干预框架，用于研究自监督视觉Transformer（ViT）如何编码密集的几何信息。虽然线性探测被广泛用于评估几何表示，但它将特征视为黑箱，无法解开底层拓扑结构。为解决这一问题，我们使用奇异值分解（SVD）分解收敛线性探测的权重，以隔离包含显式几何信号的低秩子空间。我们的视角产生了三个关键见解：（1）预训练目标决定了特征的编码方式。DINOv2对齐空间特征以实现高效的线性提取，而掩码自编码器（MAE）倾向于分散这些信号，需要更广阔的空间上下文。（2）显式几何表示高度可压缩，这表明密集预测头可能被限制在低秩子空间中，性能损失极小。（3）逐层任务亲和性表明，几何精度在中间层达到峰值，然后在最终层让位于语义抽象。通过连接内部编码机制与下游性能，这些发现为有效的特征选择和轻量化解码器设计提供了基础。源代码可在 https://github.com/Zhou-Weichen/Geosubprobe 获取。

## Abstract
We introduce a controlled subspace intervention framework to investigate how self-supervised Vision Transformers (ViTs) encode dense geometric information. While linear probing is widely used to assess geometric representations, it treats features as a black box, failing to disentangle the underlying topology. To address this issue, we decompose the weights of converged linear probes to isolate the low-rank subspaces containing explicit geometric signals using Singular Value Decomposition (SVD). Our perspective yields three key insights: (1) Pre-training objectives determine how features are encoded. DINOv2 aligns spatial features for efficient linear extraction, while Masked Autoencoders (MAE) tend to disperse these signals, requiring a broader spatial context. (2) Explicit geometric representations are highly compressible, suggesting dense predictive heads could potentially be constrained to low-rank subspaces with minimal performance loss. (3) The layer-wise task affinity suggests that geometric precision peaks at intermediate layers before yielding to semantic abstraction in the final layers. By connecting internal encoding mechanics with downstream performance, these findings provide a basis for effective feature selection and lightweight decoder design. The source code is available at https://github.com/Zhou-Weichen/Geosubprobe.

### 核心思想
提出一个**受控子空间干预框架**，利用奇异值分解（SVD）分解线性探测器的收敛权重，隔离出**任务对齐的低秩子空间**，从而量化几何信号在特征空间中的分布密度和可压缩性。

### 关键技术细节
- **步骤1：可读性差距（Readability Gap）**
  - 设计三级探测层次，从线性探针（显式几何）到MLP（局部非线性纠缠）再到DPT解码器（全局空间聚合），通过性能差距分离“几何信息是否存在”与“几何信息是否可线性访问”。
- **步骤2：子空间干预（Subspace Intervention）**
  - 对收敛的线性探针权重矩阵 \(W \in \mathbb{R}^{C \times D}\) 进行SVD：\(W = U \Sigma V^\top\)，其中 \(V\) 的列构成任务相关标准正交基。
  - 定义**对齐子空间** \(S_k = \text{span}\{v_1, \dots, v_k\}\)（k ≤ C），将特征投影到该子空间，并用**原始冻结的线性头**进行评估，所有性能变化仅归因于所选子空间的几何容量。
  - 引入**控制子空间**：随机子空间 \(R_k\)（保留维度但破坏任务对齐）和**正交补空间** \(S_k^\perp\)（残差），用于验证几何信号是否集中在前k个主方向。
- **公式与算法流程**：
  - 特征投影：\(\tilde{Z}_k = Z V_k V_k^\top\)
  - 残差特征：\(Z_{\text{res}} = Z - Z V_k V_k^\top\)
  - 能量分配：定义谱加权能量 \(E_i = \frac{\sum_{m} \sigma_m^2 \|v_m^{(l_i)}\|_2^2}{\sum_{j} \sum_{m} \sigma_m^2 \|v_m^{(l_j)}\|_2^2}\) 以量化每层对全局几何信号的贡献。

## 6. 论文的主要结论与发现

1. **预训练目标决定编码格式**：
   - DINOv2将几何信息高度对齐到线性可访问的低秩子空间（线性探针 SA-δ1=0.9157）。
   - MAE分散几何信号到更广维度，需全局聚合才能有效提取（线性探针仅0.6033，DPT提升至0.7022）。
   - iBOT为中间状态，平衡线性可访问性与局部精细度。
2. **几何表示高度可压缩**：所有模型的显式几何信号均可通过前k=32~64个奇异方向恢复>98%的线性潜力，证明高度冗余。
3. **层间任务亲和性**：
   - 几何精度在中间层（DINOv2的层18~21）达到峰值，随后让位于语义抽象（层24语义最优）。
   - 法线、深度、语义的峰值层依次后移，提示需要进行深度感知的特征路由。
4. **子空间稳定性**：低秩核心方向对随机初始化高度稳定（相似度>0.93），排除了优化伪影。
